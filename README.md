This project is a full-featured blog application that includes a powerful admin dashboard for managing content efficiently. It allows administrators to create, edit, delete, and organize blog posts while providing a clean and responsive user interface for readers.

The application is built using React for the frontend, Firebase for backend services like database and authentication, and Tailwind CSS for modern styling.

The admin dashboard includes advanced features such as rich text editing using Quill, post status control (draft, published, pending), category and tag management, pagination, filtering, sorting, and bulk actions like deleting multiple posts.
